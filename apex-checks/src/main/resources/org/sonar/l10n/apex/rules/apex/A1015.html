Assertions should have boolean variables only.

Noncompliant Code Example

@isTest
public class sampleClass() {
    public static testMethod void method () {
        ...
        system.assert(3 + 2, "message");
    }
}

Compliant Solution

@isTest
public class sampleClass() {
    public static testMethod void method () {
        ...
        boolean booleanValue = otherClass.booleanMethod();
        system.assert(booleanValue, "message");
    }
}

@isTest
public class sampleClass() {
    boolean booleanValue;
    public static testMethod void method () {
        booleanValue = otherClass.booleanMethod();
        ...
        system.assert(booleanValue, "message");
    }
}
